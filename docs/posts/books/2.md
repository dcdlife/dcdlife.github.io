# 「拆书 2」《明智创富指南》（如何成为更富更强的人）

<img style="margin: 20px auto 0 auto;" width=30% src="https://img3m2.ddimg.cn/95/12/29777612-1_u_1726114695.jpg"  >

```
好好做自己，是世间最好的生意。
```

这本书集合并梳理了彪悍一只猫老师的 2022 到 2024 年所有的创作内容，需要我们边看边停下思考。这里有两个动作供你参考：

1. 每看一页，拟定一个标题，并用最短的词进行概括，这个动作要的是我们的参与。
2. 在文字下方，写下感悟或计划。

以上这两个动作的核心是“输出”，只有输出才能**让认知从你的内在真正生长出来**。

下面是我的输出，你无论如何也要做出自己的输出。当某一天你回顾自己的输出，发现有个点不太清晰时，这时候就再去找书中对应的地方，重新再看，那么就这一个点就会在你的脑海中存在很长的时间（这就是温故而知新的重要性）。

- p1

  彪悍一只猫在回答“你的内容是写个哪些人看的”时说：那些“极度渴望变富、变强并愿意积极行动的人”。

  - 主题：为什么要写？
  - 感悟：对我来说，我想写作的主要目的，就是提升自己，不断不断的提升自己，都说“好记性不如烂笔头”，身边的很多例子都证实了这句话，所以无论现在你我有多么穷困潦倒，我觉得去读书和写作都能帮助我们蹚出一条新的路子。

- p2
  - 主题：打造 IP
  - 感悟：应该很多人都想打造自己的 IP，并想以此来赚取收益，这没错，我也如此。但如文中所说，更重要的是要修己达人，只有先做好自己，把自己打造好，才能更好地传授方法和经验，从而渡己渡人。
- p3
  - 主题：学会售卖
  - 感悟：这个“学会售卖”，不是单纯说卖东西，往深了理解，说的是卖价值，要懂得人性，要懂人们需要什么，什么是对自己对他人有价值的东西，这个东西又能如何影响他人，是不是会影响别人变的更好，如果这些问题的答案都是**好的**，那么你的这个东西自然就会产生很好的收益，甚至还会有更多的人去推荐。
- p4
  - 主题：互利共赢
  - 感悟：这世上有太多强大的人和物，我们都无法与之抗衡，那索性就不要抗衡，去看看能不能为我所用，能不能互利共赢。
- p5
  - 主题：学会拒绝
  - 感悟：无论你在工作还是生活中，都会有来自很多人的打扰，当然有的是正向的有的可能是负向的，但无论是什么打扰，我们都要本着为自己的时间和精力负责的角度，进行考量，合理恰当的拒绝不必要的打扰，这里文中提到了一个技巧，就是示弱，当别人觉得你不容易时，也就更容易接受你的拒绝。（这里想到一个小例子，当别人约你打球时，这时候你已经有别的安排必须要做，这时候便可以身体为由，委婉恰当的拒绝）
- p6
  - 主题：管理预期
  - 感悟：无论你要交付什么，都要管理好预期，不仅自己要明白交付的结果是什么样，也要让接收者明天拿到的是什么样，从而尽量避免给自己和接收人带来困惑。
- p7
  - 主题：分享致谢
  - 感悟：做成一件事，把经验分享出来，既帮助自己整理也可以帮别人进步，如果受到了别人的帮助才做成的，那更应该致谢，这样以后他人也会更乐于再次帮助。
- p8
  - 主题：和高人学
  - 感悟：这页和知识星球中墨问的专栏《让时间为你证明》中的和高人聊主题一致，都是说要和高人一起学，每个人在刚开始学的时候，都是盲人摸象，都有自己的理解，如果这时候学到了高人的见解，则可以更好的促进自己的理解。
- p9
  - 主题：屏蔽干扰
  - 感悟：这页和 p5 类似，p5 讲学会拒绝，这里说的是学会聚神，让自己保持专注，我们很多人的朋友圈，很多 App，甚至有些人都是不值得去花时间的，要让自己的时间聚焦在正确和值得的事情上。
- p10
  - 主题：营造环境
  - 感悟：环境对人的影响是巨大的，我们一定要去挑选有利于自己成长的外部环境，什么样的环境和以及处在这个环境中的人会反过来影响并塑造我们。张小龙在《微信的产品观》一书中就提到：人是环境的反应器。
- p11
  - 主题：加速成长
  - 感悟：如果你成长的更快，那就会比别人更快的获得成长带来的各种收益。一个是跟领域内的高手学习，高手只所以是高手，自然是有道理的，站在巨人的肩膀上学习，自然也就比别人看的看的更远。再一个是多读书，重点是对应领域的经典著作，读书是任何学习都离不开的不二法门。最后就是实战，要尽可能的多去实践，实践会遇到问题，这些问题对我们来说都极其宝贵。总结下来就是：跟高手学；读经典著作；实践解惑。
- p12
  - 主题：高效成长
  - 感悟：成长分快和慢，分别对应高效和低效。任何人都想要高效的成长，那如何高效成长呢？我觉得这是个很宽泛的问题，涉及的解答也太多了，很多方面都可以影响高效的成长，这里书中给到需要去大量面对真实的人，真实的问题，这句话中包含 2 个关键的点：大量和真实。大量很容易理解，真实我理解说的就是要去面对和应对一切真实的东西，只有真实，我们花的时间才是花到对的地方了。这个真实是可落地的，任何你解决的问题都能够落地，落成成果，比如：落地成文章、落地成照片，也就是落地成一切你以后可以追溯的东西。
- p13
  - 主题：负责自己
  - 感悟：很简单，别指望别人对你负这个责，负那个责，你才是自己的最终责任人，你才是自己时间的 CEO。
- p14

  - 主题：时间管理
  - 感悟：这页讲的是对自己的时间管理，时间管理是个老生常谈的问题了，之前已经学到了粥左罗老师的《时间管理课》，讲的很好，今天读到这个每天给自己一段不被打扰的时间，去输入和输出，又有了新的感觉。一件事情持续几天，几个月可能并不见成效，但以年为维度，那绝对是会让你可怕的成长。这么想想，如果你找到一件你非常非常想做的事情，每天给自己不被打扰的半个小时，一年之后的你，肯定会被自己的成长而吓一跳。时间管理很宏达，可以从每天给自己不被打扰半个小时开始做起。

- p15

  - 主题：推陈出新
  - 感悟：毫无疑问，任何一个东西拥有久了，你我都会去想换新的，如果你有自己的产品或者自己的创作等等，用户用久了之后也会有这个想法，所以要经常推陈出新。

- p16

  - 主题：互补
  - 感悟：何为互补，就是你不只需要 ta，ta 也需要你，ta 感觉到你很有价值，于此同时也要让 ta 感受到你很需要 ta，这样的关系才能更长久。

- p17

  - 主题：共赢
  - 感悟：这页和 p4 一样，都在强调共赢，什么事情你都自己一个人做，并不能说明你有多强，反而是弱的体现，俗话说的好，一个人走的快，一群人走的远，要想走的更远，到达更远的地方，学会共赢。

- p18

  - 主题：亲近客户
  - 感悟：现如今加入你有自己的虚拟产品，那和客户的交流更多是在线上，然后如果能更近一步，来到线下，你与客户的信任可能会建立的更快。

- p19

  - 主题：学会珍惜
  - 感悟：如果一个人连自己都不知道珍惜，那又怎么能过好这一生呢，珍惜自己，不仅要珍惜身体，也要珍惜你那与生俱来的优势，更要珍惜你身边的人，珍惜那些跟你合得来的，那些能给你提供帮助的人，这样的你是又是多么富有啊。

- p20

  - 主题：屏蔽干扰
  - 感悟：和 p9 一样，都是说的要屏蔽干扰，如果你所在的环境无法让你专注时，那么就要主动去选择环境，图书馆，咖啡馆，自习室等等任何能让你自在做事的地方。

- p21

  - 主题：打造 IP
  - 感悟：和 p2 一样，这次除了修自己之外，还增加了两个点：做内容和带人，对应的思路是，做内容是通过你的创作来收获你的客户或者说粉丝，带人是指通过主动影响一波人，从而这些人可以带动更多的人。

- p22

  - 主题：爱自己
  - 感悟：还是那句话：你是自己的最终责任人，你是自己时间的 CEO。所以一定要爱惜自己的身体，累了就休息，千万不要把自己累垮，否则你所有的努力都将白费。

- p23

  - 主题：推陈出新
  - 感悟：和 p15 一样，只有不断的推陈出新你才能进步，我觉得有两个方向，一个是迭代，一个是新增。迭代我想说的是旧的东西也要经常更新，很少的事情是一成不变的，因此迭代也可以进步；新增，就是打造新的东西，毫无疑问，新增的挑战更大，同时给你带来的成长也大。所以如果想加速成长，就要不断带来新的东西，但与此同时也要对之前的东西保有迭代。

- p24

  - 主题：保持开放
  - 感悟：和 p18 亲近客户差不多，这个保持开放指的不仅是客户，任何人都可以，要多和其他人交流，沟通创造价值。

- p25

  - 主题：学会拒绝
  - 感悟：和 p5 一样，不可以随便被人叫去做这个做那个，比如：吃饭、喝酒、打球等等，你是自己时间的 CEO，要为自己负责。

- p26

  - 主题：抓重点
  - 感悟：你是自己时间的 CEO，不要把时间花费在太多种类上，人生只要抓住重点，把重要的事情做好，不求量，只求质。

- p27

  - 主题：做正事
  - 感悟：无论你当下是什么地位什么处境，对待手上的事情，都要把它做正确做好。

- p28

  - 主题：做正事
  - 感悟：和 p27 一样，这里强调的是排除干扰。

- p29

  - 主题：做正事
  - 感悟：和 p27、p28 主题一致，这里强调带人，给优秀的人更好的发展空间，更好的待遇。

- p30

  - 主题：做正事
  - 感悟：和前几页一样，主题都是做正事，这里针对的方向是社群，对这部分人要严格筛选，并认真交付。（值得思考）

- p31

  - 主题：做正事
  - 感悟：依然是做正事，这里针对的方向是自己和家人，你既要爱自己的家人，也要爱惜自己的身体。

- p32

  - 主题：待人
  - 感悟：现如今做什么都讲究口碑，好的口碑，会口口相传，为你带来更多。举例来说：拿距离周边的人的远近比喻洋葱的圈层，你是中心，那如果你对自己最近的这一层的人好，那么久而久之，外层的也会逐渐被你吸引到。

- p33
  - 主题：做正事
  - 感悟：这页我也把它归到了做正事，但这里说的正事本身，一开始的我们都不知道做什么正事，不知道定位在哪，但是随着你不断做事，它会逐渐明了。
